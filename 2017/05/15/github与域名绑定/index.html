Posted on 2017-05-15

Github 和 个人域名绑定

  • 第一步,在阿里云购买自己的域名,在域名列表中选中需要绑定的域名解析,如下图:

  • 第二步,填写你的IP解析，此处的IP是你github的,如我的是paul15342.github.io


    然后添加一条解析记录类型是CNAME,记录值是paul15342.github.io


  • 第三步,登录GitHub，进入之前创建的仓库，点击settings，设置Custom domain，输入你的域名,保存


  • 第四步, 进入本地博客文件夹 ，进入hexo/source目录下，创建一个记事本文件，输入你的域名，只填写域名即可,保存，命名为CNAME ，注意保存成所有文件（不带任何后缀）



    然后在git bash中一次完成下面三个指令

    hexo clean

    hexo g

    hexo d

打开浏览器输入你的域名，即可进入到你的博客页面
